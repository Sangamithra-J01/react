# React + Vite Starter

This repository contains a minimal React app bootstrapped with Vite.

## Prerequisites

- Node.js 18+ (recommend LTS). Verify with:

```powershell
node -v
npm -v
```

## Install & Run (Windows PowerShell)

```powershell
# Install dependencies
npm install

# Start dev server (opens on http://localhost:5173)
npm run dev

# Build for production (outputs to ./dist)
npm run build

# Preview the production build locally
npm run preview
```

## Project Structure

- `index.html`: App entry HTML
- `src/main.jsx`: React entry with `createRoot`
- `src/App.jsx`: Example component with a counter
- `src/index.css`: Basic styles
- `vite.config.js`: Vite configuration
- `package.json`: Scripts and dependencies

## Notes

- The dev server is configured to open automatically on port 5173.
- Update `title` in `index.html` to your app name.
